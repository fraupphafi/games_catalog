# Games Catalog

Simple application for view games list.
Made with React Native and TypeScript.

## How to run
```
  yarn
  npx pod-install

  yarn ios _or_ yarn android
```

## Stack
List of used libraries

- react navigation
- react native paper
- react query
- ky
